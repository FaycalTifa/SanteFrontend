Import des médicaments

Importez le fichier CSV contenant la liste des médicaments et leurs exclusions

Format du fichier CSV :
  • Colonne A : NOM du médicament
  • Colonne B : EXCLUSION (OUI ou NON)
  • La première ligne doit contenir les en-têtes "NOM;EXCLUSION"
  • Séparateur : point-virgule (;) ou virgule (,)

Import des examens

Importez le fichier CSV contenant la liste des examens et leurs validations

Format du fichier CSV :
  • Colonne A : CODE de l'examen
  • Colonne B : NOM de l'examen
  • Colonne C : VALIDATION (OUI ou NON)
  • La première ligne doit contenir les en-têtes "CODE;NOM;VALIDATION"
  • Séparateur : point-virgule (;) ou virgule (,)
